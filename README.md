# Clear Lake Monitoring

Clear Lake water quality and level monitoring

## Data Update Frequency
Data is updated every 15 minutes

## Gage Stations

- **Cyrus Brouse** (cyrus_brouse)
- **Harry Tetters** (harry_tetters)
- **Clear Lake Outflow** (outflow)

## Data Access
- Raw data files are in the `data/` directory
- Statistics and metadata are in the `metadata/` directory
- Configuration is in `config.yaml`
